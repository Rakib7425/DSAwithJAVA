Definition/ Description list

HTML
- This is th markup language for web pages. It defines the meaning and structure of web content.
CSS
- This is the language used to style the HTML document. It describes how HTML elements should be displayed.
JavaScript
- It is the programming language of the Web. It is used to program the behaviour of web pages.
Python
- It is used on a server side to create web applications.
